## Project Title

The **Movie Watchlist Application** is a simple web application that allows users to create and manage a list of movies they want to watch. Users can easily add movies to their list and remove them as they watch. 

## Description

- **Add Movies**: Enter a movie name and click the **"Add Movie"** button to add it to your list. You cannot add the same movie more than once, as the application prevents duplicates.

- **Remove Movies**: Click the delete icon next to a movie to remove it from the list.
